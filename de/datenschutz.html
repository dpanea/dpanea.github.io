3. Erhobene Daten

Server-Logs

Beim Besuch der Website können durch den Hosting-Anbieter technische Daten wie IP-Adresse, Browsertyp, Referrer und Zeitstempel verarbeitet werden. Diese Daten dienen Sicherheit und Betrieb der Website.

Kontakt und Scorecard

Wenn Sie Kontakt aufnehmen oder die Scorecard anfordern, werden die von Ihnen eingegebenen Daten zur Bearbeitung Ihrer Anfrage oder zur Zustellung der angeforderten Inhalte verarbeitet.

Cookies und Local Storage

Diese Website ist so gestaltet, dass keine Tracking- oder Werbe-Cookies eingesetzt werden. Eine manuell gewählte Sprache kann im Local Storage gespeichert werden.

Analytics

Diese Website kann Cloudflare Web Analytics für aggregierte, cookie-freie Messung von Seitenaufrufen, Referrern, Browserinformationen und ungefährer Länderstatistik nutzen. Es findet keine individuelle Besucherprofilierung statt.
