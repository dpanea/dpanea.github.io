Política de Privacidad

Última actualización: abril 2026

1. Resumen

La protección de tus datos personales es importante. Esta política explica qué datos se tratan cuando visitas el sitio web, solicitas la scorecard o contactas conmigo.

2. Responsable del tratamiento

3. Datos recogidos

Logs del servidor

El proveedor de alojamiento puede recoger datos técnicos como dirección IP, navegador, URL de referencia y fecha/hora para seguridad y funcionamiento del sitio.

Contacto y scorecard

Si contactas conmigo o solicitas recursos, trato la información que facilitas para responder a tu consulta o entregar el contenido solicitado.

Cookies y localStorage

Este sitio está diseñado para evitar cookies de seguimiento. La preferencia de idioma puede guardarse en localStorage cuando eliges un idioma manualmente.

Analítica

Este sitio puede usar Cloudflare Web Analytics para medición agregada y sin cookies de páginas vistas, referencias, navegador y país aproximado. No está pensado para perfilar visitantes individuales.

4. Base legal

El tratamiento se basa, según el contexto, en consentimiento, ejecución contractual o precontractual, obligación legal o interés legítimo conforme al RGPD.

5. Conservación

Los datos se conservan solo durante el tiempo necesario para la finalidad correspondiente o mientras existan obligaciones legales. Las consultas de contacto suelen conservarse hasta tres años salvo solicitud de eliminación anterior.

6. Tus derechos

Puedes solicitar acceso, rectificación, supresión, limitación, portabilidad u oposición al tratamiento de tus datos. Para ejercer estos derechos, usa el enlace de email protegido en esta página.

7. Proveedores externos

El sitio puede usar GitHub Pages para alojamiento, Cloudflare Web Analytics para analítica agregada sin cookies, Calendly para reservas y ConvertKit para entrega de la scorecard. Selecciono proveedores teniendo en cuenta requisitos de protección de datos.

8. Transferencias internacionales

Algunos proveedores pueden tratar datos fuera de la Unión Europea. Cuando sea necesario se aplican mecanismos adecuados, como cláusulas contractuales tipo u otras garantías equivalentes.

9. Cambios y reclamaciones

Esta política puede actualizarse. Si tienes preguntas o reclamaciones sobre privacidad, contacta conmigo. También puedes reclamar ante la autoridad de protección de datos competente.
